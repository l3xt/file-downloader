# File Downloader

Утилита командной строки для загрузки файлов, написанная на Go.

## Описание

File Downloader - это простой и эффективный инструмент для загрузки файлов из Интернета с поддержкой различных протоколов и возможностей.

## Возможности

- Быстрая загрузка файлов
- Отображение прогресса загрузки
- Поддержка возобновления прерванных загрузок
- Работа с различными протоколами (HTTP/HTTPS)
- Сохранение файлов с автоматическим определением имени
- Конкурентная загрузка нескольких файлов

## Установка

### Предварительные требования

- Go 1.21 или выше

### Установка из исходного кода

```bash
git clone https://github.com/l3xt/file-downloader.git
cd file-downloader
go build -o file-downloader main.go
```

## Использование

### Синтаксис

```bash
./file-downloader <путь_к_директории> <URL1> [URL2] [URL3] ...
```

Первый аргумент - это путь к директории для сохранения файлов, все последующие аргументы - URL-адреса файлов для загрузки.

### Примеры

```bash
# Загрузка одного файла в текущую директорию
./file-downloader . https://example.com/file.zip

# Загрузка нескольких файлов в указанную директорию
./file-downloader /path/to/directory https://example.com/file1.zip https://example.com/file2.pdf
```
